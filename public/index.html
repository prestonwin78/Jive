Simple, Lightweight, Free

Jive is a simple and easy to use task manager which allows users to easily add items to their to-do list, change the completion status of items, and customize the look of the application all within the website.

How It Works

Jive is a personal daily planner website in which users can set daily tasks and see task completion on a daily basis. The user can see what tasks have been accomplished, what has been postponed, and what was cancelled. The website shows each task for the week organized by day.

Simple Design

Tabs for all of the days of the week to the user and provide easy access to different weeks.

Easy Access
